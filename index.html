structure

< html >

<!DOCTYPE html>
<html>
 <head>
 <title>Website Title</title>
 </head>
 <body>
 <p>Makin' Bisquits</p>
 <footer>
  <Website Footer>
 </footer>
 </body>
</html>

{ css }

h1 {
font-family: VT232;
font-weight: bold;
}

.class {
display: inline-block;
background-color: #1b1b1b;
}

#id {
color: orange;
text-align: center;
}

Here is some basic boilerplate html and a mini example of css structure. If we take a look at the structure of html, it's pretty comparable to word or google docs. Reading from top to bottom, we see the doc type declaration and language (html) declaration. Beneath that we have our header, not unlike the header you'd find in word. Within that is the title of the website, this is what appears in your broswers tab.
